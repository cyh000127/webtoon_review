# Webtoon Archive

이 폴더는 사용자가 제공한 웹툰 감상 목록을 공식/공개 웹 정보 기준으로 정리한 로컬 아카이브입니다.

- `webtoons.xml`: 요청한 XML 데이터
- `webtoons.json`: 나중에 앱 데이터로 변환하기 쉬운 보조 JSON
- `covers/`: 작품 표지 이미지
- `description/`: 작품별 소개 텍스트

XML에는 source URL을 넣지 않았습니다. 일부 입력 제목은 공식 작품명 기준으로 보정했고, 보정 내용은 각 항목의 `note`에 남겼습니다.

## 현재 구성

- 웹툰 항목: 72개
- 표지 이미지: 72개
- 작품 소개 파일: 72개
- 수집 기준일: 2026-05-25
- 마지막 갱신일: 2026-05-31

## 감상 상태 값

- `reading`: 읽는 중
- `finished`: 완주
- `dropped`: 중도 포기

## 검증

아카이브를 수정한 뒤에는 아래 명령어로 XML, JSON, 표지, 설명 파일 매칭을 확인합니다.

```bash
npm run validate:webtoons
```

검증 항목은 다음과 같습니다.

- XML의 `count`와 JSON 항목 수 일치 여부
- XML 웹툰 항목 수와 JSON 항목 수 일치 여부
- JSON id 중복 여부
- 필수 필드 존재 여부
- 사용자 평점 범위 여부
- 표지 파일 존재 여부
- 설명 파일 존재 여부
